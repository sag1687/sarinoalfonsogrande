Repository plugin QGIS — Dott. Sarino Alfonso Grande

Per installare i plugin, aggiungi questo repository in QGIS:

  1. Apri Plugins → Gestisci e installa plugin → Impostazioni
  2. In Repository dei plugin premi Aggiungi
  3. Nome: SinoCloud — URL: https://sag1687.github.io/sarinoalfonsogrande/plugins.xml

Plugin disponibili

  • Q-Press v2.1.1 — IT: Genera PDF cartografici professionali dall'area selezionata sul canvas con Shift+trascina: scala, griglie, legenda, tabelle e profilo topografico. (zip)
  • Quick CRS Fixer v2.1.1 — IT: Rileva e corregge automaticamente i problemi di sistema di riferimento (CRS) dei layer, con suggerimento EPSG intelligente. (zip)
  • GeoBridge v1.3.0 — IT: Client non ufficiale per le API IGM: conversione di coordinate e layer vettoriali tra datum italiani. (zip)
  • QGIS Ledger v4.2.1 — IT: Controllo di versione stile Git per progetti QGIS: diff geometrici, rollback e sincronizzazione multi-cloud. (zip)
  • GeoCSV Mapper v2.3.1 — IT: Importa CSV con coordinate in vari formati (incluso DMS), anteprima mappa e salvataggio in GeoPackage. (zip)
  • STAC Browser v1.7.0 — IT: Cerca e scarica dati di osservazione della Terra dai cataloghi STAC disegnando un'area o cercando un indirizzo. (zip)
  • SARIAG v0.3.1 — IT: Serie temporali InSAR da Sentinel-1 con SNAP: mappe di spostamento Est-Ovest e Verticale. (zip)
  • TAF Italia v2.7.1 — IT: Scarica e converte i Punti Fiduciali (TAF) dell'Agenzia delle Entrate in CSV/GeoPackage WGS84. (zip)
